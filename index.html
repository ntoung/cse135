Purpose Quiz Maker Question Design Quiz Taker / Preview


  • Application is designed to facilitate the creation and administration of quizes for a classroom environment.
  • Offers easy to use drag and drop interface.
  • Options for different question types
  • On the fly preview functionality while designing a quiz.


Basic Layout


The existing wireframe with CSS includes the ability to add different types of questions to a quiz on a per-question basis. Individual characteristics may be altered to an added question, depending on the question type. The entire design process is conducted by dragging and dropping with the mouse and entering the question/answer text with the keyboard. To add a question, the Maker simply drags a question type from left-hand menu, and drops it into the highlighted area. The dropped item will expand into the necessary fields to make a question of its type. In addition, the administrator is also able to change any constrains of the quiz in the jumboron subheader.


Toolbar Design


  • Design was provided by "Flatstrap"
  • Toolbar on the left stays fixed on the browser page when scrolling
  • Design allows user to see questions and options at all times

Jumbotron subheader


  • Located at the top screen, underneath the navigation bar
  • Quiz administrator can change constraints of the quiz
  • Provides quiz options with special features:
    • Load Saved Quiz
      • Option to load a previously saved quiz
    • Quiz Name
    • Time Limit
      • Only allows integer input
      • Does not allow text input
      • Indicates non-valid input with red borders
    • Possible Points
      • Only allows integer input
      • Does not allow text input
      • Indicates non-valid input with red borders
    • Option for Revealing Answers
      • Indicates the amount of points possible for the entire quiz
      • Drop-down box with options of Never, After Deadline, or On Turn-in
    • Option to Randomize Each Test
      • Indicates whether or not to randomize questions
      • Drop-down box with options of Fixed Order or Randomized
    • Quiz Open Date and Time
      • Can manually type in date or select from drop-down calendar
      • Can select AM or PM
    • Quiz End Date and Time
      • Can manually type in date or select from drop-down calendar
      • Can select AM or PM
  • User can change attributes with arrow buttons and drop-down menus


Question Design



Main Features

  • Maker has the option of selecting different types of questions
  • Each new question added is slide into the top
  • Can drag and drop question labels on the sidebar to reorder
  • Option to reorder questions by type
  • Help buttons to give user a tour of the site

True/False

  • Enter question text and select correct answer (True or False)
  • Taker will see True or False options underneath the question and mark the answer

Multiple Choice

  • Enter question
  • Dynamically add as many possible answers as desired with "Add Answer" button (Default: two answers)
  • Option of one correct answer or multiple correct answers
  • Taker will see question displayed and select answer using radio buttons

Matching

  • Enter a list of word-pairs into a set of text input boxes
  • Taker will match words

Fill-in

  • Quiz maker enters text into the box with underscores where the blanks should be
  • Taker will enter the word(s) for the missing blank

Short Answer

  • Maker will enter the question into the question box
  • Maker can optionally specify maximum number of characters
  • Taker will fill in the text area

Inserting pictures/graphs into quiz

  • Optionally, each question type can have a picture associated with it that will be displayed to the Taker after the question text.


Backend will handle user management, scoring, and overall grading. Backend will also deal with saving Maker’s quiz, and Taker’s answers. (For text based answers, Maker needs a way to manually grade the answers and add score into calculated score)
